# Implementar script Python para o Perceptron (Regra de Hebb)

Criar o script Python para classificar a pureza do óleo a partir dos dados do processo de destilação.

## Tarefas
- [x] Mapear o dataset fornecido (30 amostras, 3 features, saídas -1 ou 1) para um formato computacional (ex: NumPy array ou lista do Python puro).
- [x] Implementar a lógica do Perceptron utilizando a Regra de Hebb com taxa de aprendizagem de 0.01.
- [x] Implementar um loop para rodar 5 treinamentos independentes.
- [x] Em cada treinamento, inicializar os pesos com valores aleatórios entre 0 e 1, garantindo que sejam diferentes entre as execuções.
- [x] Registrar ou exibir os resultados de cada treinamento.
