# Project Requirements

## REQ-001: Critérios de Treinamento do Perceptron
- **Descrição**: O algoritmo de classificação do óleo deve utilizar um Perceptron supervisionado com a regra de Hebb.
- **Taxa de Aprendizagem**: 0.01.
- **Inicialização**: Os pesos iniciais devem ser aleatórios entre 0 e 1.
- **Execução**: O modelo deve ser treinado 5 vezes (rodadas independentes). Para cada rodada, a inicialização deve garantir pesos iniciais diferentes.
- **Dados**: Conjunto de 30 amostras com 3 características (x1, x2, x3) e 2 classes (C1 = -1 e C2 = 1).
